Provides interfaces for accessing platform extension definitions.

Platform provides the following extension points:

  • platforms - platform definitions, e.g. S60 3.0, 3.1, etc.
  • devices - display information, size and orientation
  • layouts - collection of layout xml files for supported devices
  • themes - theme subsystems

The active contributions can be read from the static methods provided in the {@link PlatformExtensionManager} class.
