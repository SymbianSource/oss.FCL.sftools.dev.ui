Outline View


The Outline view provides a structured view of all the components within the screen displayed in the editor. This view provides a useful way of accessing all themeable components in a screen.


Figure: Outline View

The Outline view has two modes which can be accessed by selecting the mode tool buttons. These modes are:
  • Containment mode ( ) – this mode displays all the components of the active screen in a tree view. You can access a component by selecting its symbol from the tree. The selected component is then highlighted in the Editor.
  • Category mode ( ) — this mode displays the components within a view by the following categories:
  • Areas
  • Background
  • Highlights
  • Icons
  • Indicators
  • Scroll Bars and Lines
  • CSS properties of Application themes
You can expand a category by selecting the expand/collapse icon (). Each category can expand into a number of sub-categories.  You can access a component by selecting its symbol from the tree. The selected component is then highlighted in the Editor.
 

Context Menu

The Outline view offers two context menus depending on whether the mouse pointer is:

  • Over a theme element.



  • Elsewhere in the editor view.




Copy Copies the selected item
Paste Paste the previously copied item
Undo Apply Graphics Undo the applied graphics on the theme
Redo Apply Graphics Redo the applied graphics on the theme
Zoom In Zoom in the selected item
Zoom Out Zoom out in the selected item
Add Bookmark... Add bookmark to current working theme
Add Task Add task to the current working theme
Select File Select a file to add an image for the selected item
Convert and Edit in Bitmap editor Coverts a *.svg to bitmap and edit the selected icon
Open in System associated program Opens a selected icon in system associated program
Edit Image in Vector editor Edits a selected image in a vector editor
Clear Clears all the previous actions
Show in Shows the selected item in the available views
