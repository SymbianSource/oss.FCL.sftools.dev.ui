Switching to 11-piece element and a single-piece element

Some components can be switched between either 11-piece graphics or single-piece graphics mode. In 11-piece mode, the component consists of  four corner graphics, two side graphics, one bottom graphic, three top graphics and a center graphic. You can design each of these graphics individually.

11 piece elements are divided into:

  • Top Left
  • Top
  • Top Right
  • Top Middle
  • Top Middle right
  • Left
  • Center
  • Right
  • Bottom Left
  • Bottom
  • Bottom Right
  1. Slice a component into 11 pieces:
    1. In the Editor or the components view, right-click on a component which is a single piece which can be broken into 11 pieces and select Switch to 11-Piece element.
      The Action confirmation window opens and prompts you to confirm if the image can be sliced and converted into a bitmap image, if required.
    2. If you want Carbide.ui to remember your decision, mark the checkbox.
    3. Click OK.

In the Layers view, you see that the image is sliced into a components consisting of 11 parts.

Note: Not all the components can be switched to 11-piece mode.

  1. Switch back to a single-piece element:
    1. In the Editor or the components view, right-click on a component which is a 11- piece which can be converted  into a single piece and select Switch to Single-Piece element.
      The Replace confirmation window opens and prompts you to confirm if the image can be replaced with the current visible image.
    2. If you want Carbide.ui to remember your decision, mark the checkbox.
    3. Click OK.

    In the Layers view, you see that the image consists of a single part.

    Note: Not all the components can be switched to single piece mode.

    Note: This is not applicable for all platforms.
