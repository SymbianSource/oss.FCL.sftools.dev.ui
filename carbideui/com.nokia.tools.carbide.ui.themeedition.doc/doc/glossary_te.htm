Glossary

Term

Definition

ASCII

American Standard Code for Information Interchange

.bmp

bitmap file format

.cer

Internet Security Certificate File

Certificate Generator

Certificate Generator is a PC-based command line tool for creating private keys and self-signed certificates.

Color group

A color group contains all the customizable colors in a theme. color are grouped into color groups according to their parent UI components. Each color is affected by adjusting the color parameters of a group.

Component

A component is an element visible in the screen of a mobile phone. Components can be icons, text boxes, or backgrounds, for example.

DRM

digital rights management

.gif

Graphics Interchange Format

GPRS

General Packet Radio Service

IrDa

A suite of protocols for exchanging data between two devices with infrared.

.jpeg

Joint Photographic Experts Group

.p7c

Digital ID file

PC

Personal Computer

PEM

Privacy Enhanced Mail

.pip

Protected information package (PIP) is Nokia's recommended DRM mechanism for S60 applications. PIP file is a ZIP archive containing the application SIS, data files to be protected, and a definition file.

.png

Portable Network Graphics

theme

In Carbide.ui, themes are referred to as themes.

Real time model

Real time model is a timing model for fast animations. By choosing the real time model, you can define the animation time in seconds.

Relative time model

Relative time model is a timing model for slow changes in an animation. By choosing the relative time model, you can select an animation to play monthly, weekly, daily, or hourly.

SDK

Software Development Kit

.svg

Scalable Vector Graphics

.sis

SIS files are installation files containing, for example, compressed themes for sending to mobile phones.

.tdf

theme design file

.tpf

A zip archive file that contains a .tdf file.

.tiff

Tagged Image File Format

UI

user interface

USB

Universal Serial Bus

WLAN

Wireless Local Area Network
