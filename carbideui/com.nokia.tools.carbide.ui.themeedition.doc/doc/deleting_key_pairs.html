Deleting Key Pairs

  • Select Tools -> Handle Key Pairs.
  • The Key pairs window opens.
  • Select a key pair by clicking its name in the Key pairs text field.
  • Click Delete.
  • The Delete key pair message box opens.
  • Click Yes.
  • The defined key pair is no longer available in the Key pairs window. However, the private key and the certificate are still in the folder where you saved them.
