Context Menu in Navigator view


The context menu  is available in the navigator view.  Right click on the navigator view , the context menu will be available. You can choose the following options.




New All Provides options to create a new theme . The project and other options are redundant in Carbide.ui
Go Into Root Changes the Navigator display to show the structure of the selected project only.
Open Editable item
Open with Editable item
Open in New Window Root Allows a project to be opened in a new window
Copy All Copy the project folder
Paste - Paste previously copied project files into selected folder
Delete - Delete a file or a project folder
Move - Move a file or folder in a project
Rename - Rename a file or a project folder
Import - Import a project
Export - Export a project
Refresh - Refresh the complete project folder
Close Project - Provides option to close project
Close Unrelated Projects - Provides option to close other projects expect for the working theme
Team - Provides option to Add a patch from a other project to a current project and also allows to Share the current working project to other teams through a project repository
Compare with - Provides option to compare with Local history of the project and each other projects
Restore from Local History - Provides option to restore changes from the local history
Properties - Displays project properties


The Navigator view contains the file structure in a tree view form for the themes opened in the workspace. It enables managing the theme file structure. Double clicking on a ".tdf" file opens the Editor. By default, the ".tdf" file opens in the UI editor. It can also be opened with the Text Editor in an XML editable format.
