Editing Key Pairs


  • Select Tools -> Handle Key Pairs.
  • Select a key pair by clicking its name in the Key pairs text field.
  • Click Edit.
  • Edit key pair details:
    • Enter a new name for the key pair, if necessary.
    • Select a new private key.
    • Select a new certificate.
    • Enter the original password for the key pair.
    • Click Save.
 Note: You cannot change the password for a key pair in Carbide.ui.
