Managing Views

You can manage the location and the size of the views in Carbide.ui. This way you can customize the tool according to your preferences. In Carbide.ui, all the possible views are not displayed automatically. In addition, some of the views are collected in tab groups, and you can access those views by clicking the name of the desired view in the tab. Some views are only accessible through the Window menu or through the Resources view. You can restore layout by selecting Window -> Reset perspective.

The following customizations are available:
  • Showing and hiding views
    • You can close the views you do not need to allow more space for the views you are using.
  • Moving views
    • You can move and tile the views according to your preferences using drag and drop.
  • Changing the view size
    • You can change the view sizes by dragging the outlines of the view.

Showing and Hiding views

  • To display a view that is not shown, in the menu bar, select Window -> Show View.
  • Click on the view you want.
Note: Some views are located under Other.

To hide a view, click X on the tab of the view.

Moving views

There are many ways to move the views.

Drag and drop
  • From the tab of a view, drag the view towards the desired location.A black arrow appears, indicating which way the editor view area splits.
  • When the arrows show the view to be in the desired location, drop the view.
Note: You can also drop the view into a tab group, so that it becomes a new tab.

Detaching a view
  • Right-click the tab of the view.
    • A pop-up menu appears.
  • Select Detached.
    • The view is detached from the work editor area and you can now drag it to a desired location
Moving a single view
  • Right-click the tab of the view.
    • A pop-up menu appears.
  • Select Move -> View.
    • The view's outline box appears.
  • Move the outline box to a desired location and click the location.
    • The view is relocated.

Moving the tab group
  • Right-click the tab of the tab group.
    • A pop-up menu appears.
  • Select Move -> Tab Group.
    • A tab group’s outline box appears.
  • Move the outline box to a desired location and click the location.
    • The tab group is relocated.
Creating a fast view
  • Right-click the tab of the view
    • A pop-up menu appears
  • Select Fast View
    • The view disappears and a symbol of the view appears on the bottom of the work editor area
  • Click the symbol to open and close the view as needed. You can change the place of the symbol, for example, from bottom to the left

Changing the view size

  • Right-click the tab of the view and select Size.
  • Choose the border you want to move, for example, Top.
  • The chosen border is high-lighted.
  • Drag the chosen border to the new location.
Tip: To quickly change the view size, place the mouse cursor on the border of the view. When a black arrow appears, you can drag the border to a new location.
