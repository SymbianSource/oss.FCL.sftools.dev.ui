Using Undo, Redo, Copy, Paste and Zoom

Undo and Redo

Undo allows you to Undo the most recent editing performed on a component. Redo allows you to Redo the most recently undone edit.

Copy and Paste

Copy allows you to copy a component you have selected in the editor. Paste option allows you to paste:
  • A previously copied component image into a selected component.
  • An image from an external source, such as a compatible image file copied from a folder or an image copied from a compatible image editing application into a selected component.

Zoom

This option allows you to control the magnification of the theme’s screen displayed in the Editor. Using the context menu or shortcut keys (CTRL+- to zoom out, CTRL+= to zoom in) the level of zoom can be set at 50%, 75%, 100% and then at 50% increments to a maximum of 400%.

The zoom level can also be set with the zoom option on the toolbar.
