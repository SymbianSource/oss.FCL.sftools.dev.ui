Working with graphical theme multimedia


You can add, edit, and clear ring tones and message alerts.

Editing sounds

  • In the Resources view, select Sounds and a sound type.
  • Click a sound icon. The Icons view containing the sound file opens.
  • Right-click the sound file and select Edit Sound in sound editor.
  • Edit the sound file.
  • Save the file and exit the editor. The file is saved in the theme.
You can specify a variety of audio elements by including sound files in your theme. The sound files replace the default sounds that the handset plays, for example, for ring tones and alerts. Some audio elements allow you to limit the play duration, which also applies to the related animation. All sound elements are edited in the Sounds tab of the Theme Editor. Deleting sounds

Note: You cannot delete default sounds. You can only delete the sounds you have created yourself.

  • In the Navigator view, right-click a sound file.
  • Select Delete.
  • Click Yes to confirm that you want to delete the sound file.
  • The file is deleted from the file system.

Adding a screen saver

Carbide.ui supports the adding of a screen saver to your theme.

To add a screen saver :
Note: As a pre-requisite, you need to have a screen saver stored on your computer. The screen saver can be a .sis, .swf, .gif or a .svg file.
  • In the Resources view, select Multimedia -> Screen Saver.
  • The Screen Saver mobile screen icon appears on the right-side pane of the Resources view.
  • In the Resources view, click the Screen Saver mobile screen.
  • In the Screen Saver component view, right-click the Screen Saver component.
  • A pop-up menu appears.
  • In the pop-up menu, select Browse for file.
  • The Screen saver - Browse for file window opens.
  • In the Screen Saver - Browse for file window, browse to the correct location of the screen saver and click Open.
The screen saver is added to your current theme. A blue tick-mark appears next to the screen saver icon in the Screen Saver component view.
