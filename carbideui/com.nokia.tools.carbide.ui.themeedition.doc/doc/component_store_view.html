Component Store view

The Component Store view displays thumbnails of the graphics used in other themes for a selected theme component. It provides you with the ability to reuse individual graphics from other themes, without the need to copy and paste or take the components from another theme.



The items displayed in the component store view are controlled by selecting the Adjust Component Store Settings tool button. This opens the Component Store preferences dialog.

For more information on effectively using the Component Store see Configuring Carbide.ui Settings.
