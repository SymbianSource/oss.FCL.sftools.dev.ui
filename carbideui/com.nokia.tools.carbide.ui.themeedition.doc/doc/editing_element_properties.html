Editing component properties

In the Properties view, you can view and edit component properties. Some of the properties are read-only and cannot be edited. You can:
  • View the official name of the component
  • View which platform versions support a component
  • Adjust the colour of a component
  • Optimize the component by changing its color depth and dithering it
  • Adjust line visibility
  • View the ID and bounds for the component
  • Adjust the stretch mode for the component

Note: You can check how the colors actually look in a mobile phone display by sending sample images to the device.
