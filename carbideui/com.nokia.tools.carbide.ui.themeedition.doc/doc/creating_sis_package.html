Packaging Themes

Creating sis-package

You can create a new sis-package of your theme using the New SIS-Package wizard. The package is saved in the current file system as a .sis file, and it can be transferred to a mobile phone.

In Carbide.ui, you can:
  • Create sis-packages of your theme
  • Sign the theme package
  • In order to sign, you can select one of the available key pairs, or set up new ones to be used during the packaging process.
  • Secure a theme with a Digital Rights Management (DRM) protection when creating a sis-package
