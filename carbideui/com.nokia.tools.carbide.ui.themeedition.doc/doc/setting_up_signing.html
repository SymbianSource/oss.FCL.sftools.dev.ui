Setting up signing


When creating a sis-package of a theme, you can sign the package with a private key and a certificate. This verifies the identity of the sender and ensures that the file has not been tampered with since it was signed.

The signing functionality consists of :
  • Creating a private key and certificate
  • Defining key pairs that are used to sign a sis-package
  • You can define several key pairs.
  • Editing existing key pairs
  • Deleting the key pairs you no longer need
