Editing Themes


In Carbide.ui, themes are edited or created in component level. This allows you detailed access to icon layers, for example. Some editing is performed in the views of Carbide.ui, but the extensive editing of a single component is mainly done with external editors. Carbide.ui provides the following editing options:

  • Using the Properties view to edit certain components of a theme directly, for example, navigation panes.
  • In the Layers view, you can:
    • edit the layers of a component
    • add effects to a layer
  • Using external editors to edit:
    • vector graphics
    • Bitmaps
    • Sounds
  • In the Animation editor, you can manage animations. This includes:
    • create frame animations
    • create layer animations
    • preview and play animations
    • morphing animations
You can use the Gallery view to activate the desired mobile phone screen in the Editor. You can select the component you want to edit from the Editor.
