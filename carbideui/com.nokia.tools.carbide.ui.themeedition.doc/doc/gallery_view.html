Gallery view

The Gallery view provides thumbnails of the customizable screens. You can control which screens are visible in the Gallery view, but a tool button allows any screen to be selected and opened in the Editor. Selecting a visible screen opens it in the Editor.



               

Figure: gallery view

Opening a screen in the editor

You can open a screen in the editor from the Gallery view in two ways:
  • By selecting a screen visible in the Gallery view.
  • Using the Select a screen tool button’s drop-down menu.

Removing a screen from the active Gallery view

You can close screens in the Gallery view by clicking the close icon at the top left corner of the screen’s thumbnail, and then open them again from the Select a screen tool button’s drop-down menu.

Tool Bar

The Gallery view tool bar provides three functions:

Select a Screen

This option displays a drop down list of the displayable screens for the active theme.

 From this list you can:

  •    Specify which screens are displayed in the Gallery view by selecting in the preferences option.
  •   Open a screen in the Editor by selecting the screen from Gallery view
Switch between User defined and full screen set This option allows you to switch the screens displayed in the Gallery view between the full set of screens and the user defined set (as defined using the Selecting a screen tool button).
Switch between Preview and Brochure layout mode

This option allows you to view the themes in:

  •  Preview mode.
  • Brochure layout mode.
Minimize Minimizes the Gallery view
Maximize Maximize the Gallery view
Close Closes the Gallery view
