Applications

The phone contains applications that have different appearances. Carbide.ui allows you to edit these appearances. The appearances of the following applications can be edited or changed:

  • Clock — This includes the appearance of clocks in the phone.
  • Pinboard — This includes the appearance of Pinboard in the phone.
  • Calendar — This includes the Calendar frames.
  • Settings — This includes the Focused and Unfocused List Frame.
  • Calculator — This includes the Paper and Display section.
  • Notepad — This includes the First Page section.
  • Music Player — This includes the different buttons and screen savers for the music player.
