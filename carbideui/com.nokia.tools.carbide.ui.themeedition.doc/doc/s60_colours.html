Colours


In Carbide.ui, the Colours section allows you to define the different colour choices of your theme components. The following components' colours, for example, can be edited:
  • Areas
  • Highlights
  • Pop-ups
  • Music Player
When you choose a colour component, for example Softkey Texts, you are presented with different colour choices for that component. Those colour choices can then be edited.
