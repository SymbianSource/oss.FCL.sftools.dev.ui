Importing New Component


You can skin 3rd party icons into Carbide.ui. This is done in Carbide.ui preferences page. To skin the icons, you need to know the Application's Unique Identification Number (UID) for the icon. See Symbian Developer Library for more information about the UID.

The imported icons are added into Application Icons / Context Pane Icons category. To view the Context Pane icons, in the Resource view, select Icons -> Application Icons -> Context Pane Icons. The Context Pane Icons component view opens, displaying the icons. The 3rd party icons that you have added appear at the end of the icon selection.

To add 3rd party icons into Carbide.ui
  • In the menu bar, select Window -> Preferences.
  • The Preferences window opens.
  • In the left-side pane, expand Carbide.ui and select Third Party Icons.
  • Third Party Icons pane opens on the right-side.
  • In the Third Party Icons pane, select Add.
  • The system displays an application UID 0x00000000, named Custom Icon 0.
  • In the Application UID field, enter the UID for the icon you want to add to Carbide.ui.
  • In the Name field, enter the name for the icon.
  • Click OK.
To delete 3rd party icons from Carbide.ui
  • In the menu bar, select Window -> Preferences.
  • The Preferences window opens.
  • In the left-side pane, expand Carbide.ui and select Third Party Icons.
  • Third Party Icons pane opens on the right-side.
  • In the Third Party Icons pane, highlight the icon you want to delete and select Delete.
  • Click OK.
