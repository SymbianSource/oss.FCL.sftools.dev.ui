Selecting components for editing

  • In the Resources view, select the desired main element and its sub-element. For example: Pop-up windows -> Call Handling Window. Different call handling window options are displayed in the Resources view as mobile screen locators, with the editable component highlighted in dark grey color.
  • Click the desired mobile screen to activate the component view. For example, if you are selecting indications for outgoing calls, select Outgoing Call Indications mobile screen locator.
  • The Outgoing Call Indications component view opens, and the corresponding mobile phone's example screen opens in the Editor.
  • Select the desired component by right-clicking it in the component view.
  • The selected component is highlighted and ready for editing.
Note: Once you have edited a component, the mobile screen locator for that component turns from grey to blue.
