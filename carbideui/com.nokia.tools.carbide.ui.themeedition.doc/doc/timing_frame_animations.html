Timing Frame Animations


In the Icons view, right-click an animated icon you want to time and select Edit/Animate selected elements.

To set how long a frame appears in the animation, do one of the following:
  • In the Animation editor, right-click a frame and select Set Animate time, and define a time for the frame.
  • Use the mouse to drag control points on the timeline.
To add a new control point, do one of the following:
  • Right-click the timeline and select Create Control Point to add a new control point. In the Create Control Point window, set a time for the control point.
  • Double-click the timeline to add a control point.
A control point is created where you double-clicked the timeline. An empty frame is added to the Frames section.

Note: The default frame duration is 160 milliseconds.

To set the same time for each frame:
  • Click the Distribute Animate Time button (the button with many empty frames).
  • The Distribute Animate Time pop-up window opens.
  • Set the time for the animation.
