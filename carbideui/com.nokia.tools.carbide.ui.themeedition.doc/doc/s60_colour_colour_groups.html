Colour groups


The Color group list contains all the customizable colors in groups. Expand a group to view the color inside it. In the list, colors are grouped according to their parent user interface (UI) components. Each color is affected by adjusting the color parameters of the group. You can also control the sub color in each color group separately. Once all the colors in a group are set and saved, the text color of the group turns blue, indicating that the setting is done and saved.
