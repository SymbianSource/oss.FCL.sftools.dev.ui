Creating and Editing layer animations

Note: This step applies to layers in highlights, backgrounds, and Navi Pane in the phone's screen.

  • In the Editor, right-click a component you want to animate and select Edit/Animate selected elements.The Animation editor opens.
  • Add a layer to the component by right-clicking the first layer and selecting Add layer.A new layer is created.
Note:You can add a background by selecting Add Background.

Note:You can toggle between views of layers displayed in the Preview section by marking and unmarking the layer checkboxes.

Add an effect to the second layer:
  • Right-click the layer.
  • Select Add effect and an effect.
Set and animate parameters for the second layer:
  • Right-click the layer and select Parameters.
  • Change the parameters of the effect.
  • Mark the checkbox next to Animate.
Create control points for the effect:
  • Right-click the effect and select Create Control Point.
  • Set a time for the control point.
  • In the Animated context area, set the parameter intensity at that control point by using a slider.
  • Repeat steps a-c to create as many control points as you need.

Note: Control points define the values of effect parameters on a specific moment in the timeline.
Note: Review the animation by clicking the green play button.
