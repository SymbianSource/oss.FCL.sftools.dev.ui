Filtering tasks in the Tasks view


  • In the Tasks view tab, click Configure the filters to be applied to this view button.
  • The Filters window opens.
  • In the User filters field, make sure that the Default box is checked.
  • You can add new filters by selecting New and entering a name for the filter.
  • In the Show items of type field, uncheck all the Type boxes except the Task box.
  • From the resources button list located in the middle, select On any Resources.
  • In the Description field,
    • Choose contains from the drop-down menu and add a description in the empty field if you want the filter to include task-specific markers like pane or area, for example, in the file name.
    • Choose does not contain from the drop-down menu. Add a description in the empty field if you want the filter to leave out task-specific markers like pane, or area, for example, in the file name.
    • If you want the filter to contain priority filtering, check the Where priority is box and the preferred priority box (either High, Normal or Low).
    • If you want the filter to contain status filtering, check the Where status is box and choose Completed or Not Completed, according to your preferences.
  • Click OK.
  • The Tasks view now displays only the filtered tasks.
Note: To remove filtering, select Restore Defaults -> OK from the Filters window
