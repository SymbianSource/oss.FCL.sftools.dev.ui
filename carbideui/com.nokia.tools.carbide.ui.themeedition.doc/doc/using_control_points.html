Using Control Points


In the Animation editor view, you can add, edit, and delete control points for animations. Control points define the points in the animation timeline when something changes. You can define as many control points as needed.

There are two ways to use control points:
  • In layer animations, control points allow you to define the intensity of an effect at a certain point in the timeline. When you click a control point of an animated effect, a slider appears and allows you to define the intensity of the effect at that control point.
  • In frame animations, control points mark the point in the timeline when a frame changes to another frame. There are as many frames as there are control points.
You can:
  • Add control points by:
  • Double-clicking the timeline
  • Right-clicking the timeline and selecting Create Control Point
  • Move control points to a different point at a timeline by dragging them with the mouse.
  • Delete control points by right-clicking them and selecting Remove Control Point.
Note: Control points can only be added to animated effects, and not at all to layers. To use control points, make sure you have checked the Animate box for the layer you wish to animate.
