Welcome

Welcome to Carbide.ui Theme Edition 1.0! The Carbide.ui is a tool for creative mobile user interface (UI) design, built according to the needs of a creative professional. With the help of this tool, you can create and edit themes for mobile phones. The following elements of a mobile phone's UI can be edited:

  • Display background
  • Appearance of highlights
  • Appearance of pop-up windows
  • Appearance of commonly used components and applications
  • Colours of the fonts
  • Style of icons
  • Animated elements
  • Sounds and music player

In Carbide.ui, the UI is scalable in order to support different resolutions and orientations. All the elements that you create or edit can be scaled to support different screen resolutions and devices.

For those tasks that require an image or sound editor, Carbide.ui Themes Edition can be linked to any external image or sound editor that runs in a PC-environment.
