Editing Bitmaps

  • In the Editor, click the component you want to edit.
    • The chosen component is highlighted.
  • Right-click the highlighted area and click Convert and edit Image in bitmap editor.
  • Select conversion settings and click OK.
    • The external image editor is launched in its own separate window.
  • Edit the image in the external editor.
  • Save the image.
  • Return to Carbide.ui.
The Editor is automatically refreshed to reflect the changes you made to the component. You can edit the component with Carbide.ui to apply the finishing touches and to adjust the resolution in accordance with the requirements of the mobile phone.

Note:  Graphics can be created or edited with the external image editor, but the actual customizing, such as layers and colors for optimal appearance for a mobile phone’s screen, must be managed with Carbide.ui.

Note: You should configure an bitmap editor to edit images. for more information see External Tools.
