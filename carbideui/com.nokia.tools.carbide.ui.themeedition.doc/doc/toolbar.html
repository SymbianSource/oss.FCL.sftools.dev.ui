Tool bar

The Tool bar, shown below, contains shortcuts for the most commonly used functions of Carbide.ui. The content of the toolbar depends on the type of theme currently displayed in Carbide.ui.

The S60 / Symbian graphical theme toolbar

From left to right the tool buttons in the S60 / Symbian toolbar are:

Create a new Theme

Save

Print

Open an existing theme

Undo and Redo

Edit/Animate Selection
Create a new sis-package

Launch Third Party Icons Definition Tool

Search a screen element

Locate the selected files on the Disk

Drop-down zoom menu
Zoom in and Zoom out

S60 platform and device selector

Resolution

Switch to Portrait or Landscape mode
