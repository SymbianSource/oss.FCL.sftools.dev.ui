Plug-in Management

Plug-in management allows you to add and remove phone graphics definition plug-ins in order to include device specific graphics. 
  • On the Menu, go to Windows -> Preferences. The Preferences windows will open (see below figure).
  • Go to Carbide.ui -> Plug-in Management.
 
Figure: Plug-in Management

To add or install a Plug-in

  • Select Add. The Install Plug-in window opens..
  • Select Browse to select the file you want to install.
Note: Plug-in files should be either in .ZIP or .JAR format.



Figure: Install Plug-in

  • Select Install button and the plug-in will be successfully installed . The installed plugin will be available in the Installed plug-ins list.


Figure: Installed Plug-in

  • Now,  you can create a new theme and change the plug-in from the  "Switch platform" option from the tool bar. This plug-in while apply changes to your new theme and can be packaged and transfered to your mobile.

Figure: Switch Platform
