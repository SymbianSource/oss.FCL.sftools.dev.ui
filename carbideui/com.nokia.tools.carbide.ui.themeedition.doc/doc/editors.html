Editors


Carbide.ui contains two editors:
  • The UI editor which displays device screens.
  • The animation editor which allows for the editing of animated components.

Editor

The editor allows you to preview device screens and select theme components. If you have several themes open, each theme is shown in its own tab in the editor. You can navigate between these themes by clicking on the tabs.


Figure : Editor
The key functions in the editor are:
  • Selecting components from the displayed screen to enable editing.
  • Editing of components and copying the image of components.
  • You can change colors of the component
These functions are available by either selecting elements in the editor or are accessed from the UI editor’s context menu.

Context menus

The Editor offers two context menus depending on whether the mouse pointer is:

  • Over a theme element.


  • Elsewhere in the editor.
