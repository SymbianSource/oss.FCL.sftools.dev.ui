Opening an existing theme


In the menu bar, select File -> Open Theme. Or from the toolbar or from context menus in navigator, you can select any project and access the folder to open a theme.
 
To select an existing theme:
  • If you want to open a theme from the current workspace, select Select a theme from current workspace and click on the theme you want.
  • If you want to open a theme from another location, select Select a theme from another location, and in the theme field either enter the location or browse to the desired location and click OK.
  • Click Open.
  • The selected theme opens in the Editor.
