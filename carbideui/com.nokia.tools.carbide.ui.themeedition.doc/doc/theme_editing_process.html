Theme Editing Process


In Carbide.ui, the themes are created by customizing the phone's default UI images and sounds. The different parts of theme creation process are listed below.
  1. Create basic theme background images and general full screen background images.
  2. Modify icon, text, and line colors.
  3. Add images that change the outlook of specific areas.
  4. Create application menu and submenu icons.
  5. Adjust list, grid, and input highlights.
  6. Define graphics for pop-up windows.
  7. Adjust miscellaneous icons and theme components.
  8. Add application-specific theme items.
  9. Add sounds and a screen saver.
  10. Create a .sis file of your theme and protect the theme using DRM. Send the theme to a separate file system or mobile phone.
