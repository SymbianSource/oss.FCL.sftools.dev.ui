Configuring the Carbide.ui settings

During the Carbide.ui installation, certain settings, such as workspace and editors, can be configured. These settings can also be edited later. When configuring Carbide.ui, you can set:

General user preferences
These preferences include, for example, confirmation, appearance and display.

Component store
The Component store allows you to access the graphics or to collect graphics for theme elements for re-use in any themes.

Example Themes
The example theme preferences allow you to add new example themes or remove old unused ones from Carbide.ui.

External editors
Carbide.ui can be configured to be used with any commercial vector, bitmap and sound editor.

Image folders
The Image folders preferences allow folders that will shown in the Select Image dialog to be defined. This allows you to access your image folders without having to browse for them each time you use Select Image.

Plug-in Management
You can manage the plug-ins in Carbide.ui with the Plug-in Management preferences. You can view what plug-ins are available, add or remove them.

Startup Tips
The Startup Tips preferences allow you to choose whether tips should be shown when Carbide.ui Series 40 Theme Edition is launched.

Third Party Icons
Third Party icons are those icons which can be created/customised by the user.  Once created you can skin these icons along with other icons in the tool. 

Figure: Preferences Window


Note: These settings can be changed in the Preferences menu of Carbide.ui. You can also import 3rd party icons into the Carbide.ui in the Preferences.
