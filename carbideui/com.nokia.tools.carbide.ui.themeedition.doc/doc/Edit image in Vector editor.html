Edit image in Vector editor 


To edit an image in the vector editor configured in Carbide.ui (see Configuring Carbide.ui Settings), select the component you want to edit and from the context menu select Edit Image in Vector editor. The image will then be opened in your vector editor.
Once you are done editing the image, save it using your editor’s standard save option. You can now return to Carbide.ui. The UI editor is automatically refreshed to reflect the changes you have made.

Note: Carbide.ui automatically converts any Scalable Vector Graphics (SVG) files saved by your vector editor into the SVG Tiny format supported by S60 / Symbian devices.
