Icons

The phone's UI contains different icons. These icons are used to denote different selections, for example, there are different icons for different menus. Carbide.ui contains the following main icons:

Application Icons
These include the Context Pane Icons, which are the icons that are used on the top part of the mobile phone's screen.
 
List Icons
These include, for example, the media gallery icons.
 
Note Icons
These include the still icons and animated icons.

Tool Bar Icons
These include the icons in the tool bar.
 
Miscellaneous Icons
These include the Find Pane Indicator icons and Browser Connecting icons. The Browser Connecting icons can be animated.

Note: Some icons can be animated. To check if an icon can be animated, in the Resources view, select Icons. When you click an icon group, the mobile screen locators appear on the right side of the Resources view. When you click one of these, it opens a component view containing the library of icons. If an icon in the library can be animated, an animation button appears on the top of the component view.

Note: When animating icons with the Animation editor, there is also a Frames section. In Frames section, you can add or delete frames into your icon animation. You cannot set effects through Properties view when animating icons.
