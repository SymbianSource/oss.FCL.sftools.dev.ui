Open in System associated program


To edit an image in the application your system has associated with the image’s file type, select the component you want to edit and from the context menu select Open in System associated program.

If your system has an associated editor for the image type, the image will be opened in that editor in a separate window.

Once you have completed the edits to the image, save it using the editor’s standard save option. You can now return to Carbide.ui. The UI editor is automatically refreshed to reflect the changes you made to the component.

Note: If there is no associated application available then by default it opens in Internet explorer.
