Bitmap conversion settings


You can convert scalable vector graphics (svg) images into bitmap images to edit them in an external bitmap editor. In order to convert images into bitmaps, you must set conversion settings in the Conversion confirmation window.

Warning: The conversion process is irreversible! It is recommended that you keep the original SVGs stored in a different location.

In the Conversion confirmation window, you can:
  • Set a preferred resolution for the image in the Bitmap resolution field
Currently selected size of element related to current editor resolution is selected by default.
  • Choose if you want to keep the image mask with the Preserve mask checkbox
  • Decide to use the selected screen resolution as default  with the Don't display this dialog again and use screen resolution as default checkbox
