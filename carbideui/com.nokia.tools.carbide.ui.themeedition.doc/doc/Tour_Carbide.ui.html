Tour of the Carbide.ui 


The user interface of Carbide.ui consists of the following elements:
  • Menu bar — The menu bar is used to access the Carbide.ui functions. You can also access the functions using keyboard shortcuts.
  • Tool bar — The tool bar contains shortcuts for the most commonly used functions.
  • Editors — Carbide.ui has two editors, one for UI screens and another for animations.
  • Panes — Panes divide Carbide.ui into areas for displaying information on the components within a theme. They contain the various views Carbide.ui offers. 
  • Views — Views provide different ways to display component and their properties, as well as view the contents of the component library. You can customize which panes the views are displayed in.
