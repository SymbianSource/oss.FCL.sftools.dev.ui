Working with Scroll bars and Lines

Adjusting line visibility

  • In the Resources view, select Scroll Bars and Lines -> Lines.
  • Select, for example, Secondary Lines.
  • In the Outline view, select, for example, Secondary Horizontal Line.
  • In the Properties view, select General:
    • Check the Draw Lines box.
    • In the Color field, select the color from the drop-down menu.
    • The Editor displays the edited line.
