Adding Effects

  • In Editor, right-click the component, whose effects you want to edit.
  • Select Edit/Animate selected elements.The Animation editor opens.
  • Right-click a layer component.
  • Select Add Effect and the effect you want to add to the layer.The effect is displayed.
Note: You can also control the effects in the Properties view.
  • Select the effect.
  • In the Properties view, the Static and Animated context for the effect is displayed. To edit the parameters of an effect:
  • Right-click the effect to display its properties in the Properties view.
In the Properties view, edit the parameters in the Static context area.The Alpha Value or the Blend Factor slider (the slider name depends on the effect) can be used to define the intensity of he effect. If you want the intensity to change during the animation, you have to animate the slider.
  • To animate the effect, in the Animated context area, select the Animate checkbox of the color you chose in previous step.
  • To define the intensity of the animated effect, set the intensity for the effect. The parameters change.
Note: You can also edit the parameters of an effect by right-clicking the effect in the Animation editor view and selecting Effect Attributes.
Note: You can also add layers and effects in Layers view.
