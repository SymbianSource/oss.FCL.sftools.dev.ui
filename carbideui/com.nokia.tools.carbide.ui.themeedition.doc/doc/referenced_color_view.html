Referenced Color View


The Referenced Colors view displays the list of the referenced colors within the theme. Referenced colors are colors which are manipulated as a group by Carbide.ui, so changing a referenced color changes all occurrences of that color.

For information on using referenced colors see Referenced colors.


Figure: Referenced Color View

By default the referenced colors view is not displayed in Carbide.ui. To turn it on select Window -> Show view -> Referenced Colors .

A new referenced color group is added by selecting the add tool button

Each referenced color line consists of:

Root Color
Derived Colors
Duplicate Button
Delete Button
