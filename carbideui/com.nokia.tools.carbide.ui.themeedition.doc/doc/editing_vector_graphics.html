Editing vector graphics

  • In the Editor, click the component you want to edit. The chosen component is highlighted.
  • Right-click the highlighted area and select Edit Image in Vector editor. The external image editor is automatically launched.
  • Edit the component in the external editor.
  • Save the component in the editor.
  • Return to Carbide.ui.
The Editor is automatically refreshed to reflect the changes you have made. Carbide.ui is converting svg files to phone supported svg-tiny files automatically.You can edit the component with Carbide.ui to apply the finishing touches and to adjust the resolution in accordance with the requirements of the mobile phone.

Note: You should configure an external vector editor to edit images. For more information see External Tools.
