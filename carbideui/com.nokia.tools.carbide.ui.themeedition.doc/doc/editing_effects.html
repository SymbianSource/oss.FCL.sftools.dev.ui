Editing Effects

  • In Editor, right-click the element, whose effects you want to edit.
  • Select Edit/Animate selected elements.
  • The Animation editor opens.
  • Select the effect.
  • The Static context area opens in Properties view.
Edit the parameters of the effect:
  • Edit the parameters in the Static context area.
  • The Alpha Value or the Blend Factor slider (the slider name depends on the effect) can be used to define the intensity of the effect. If you want the intensity to change during the animation, you have to animate the slider.
  • To animate the effect, in the Animated context area, select the Animate checkbox of the color you chose in previous step.
  • To define the intensity of the animated effect, set the intensity for the effect.
