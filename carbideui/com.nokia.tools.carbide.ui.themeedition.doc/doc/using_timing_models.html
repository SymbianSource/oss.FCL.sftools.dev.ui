Using Timing Models

A timing model allows you to set when an animation is played. You can choose from two timing models:
  • Relative
    • Relative time changes are slow. It is recommended to use them in background components. The animation starts when a theme is activated in a mobile phone. Options for the relative time model are:
      • Monthly
      • Weekly
      • Daily
      • Hourly
Note: By setting the time slice to 5 and selecting Daily, the animation is shown 5 times during one day.

  • Real time
    • Real time is used for fast animations. It is recommended to use it for list, grid and focus highlight. Real time units are in seconds.
