Component store


The Component store allows you to access the graphics and to collect graphics from theme elements for re-use in any themes. For more information on using the component store see Working with the component library.

This preference dialog allows you to determine which components are displayed in the Component Store view. The available options are:
  • Show components of themes in the current workspace shows the components from the themes in the current workspace – see Choosing / switching a workspace for more information on workspaces.
  • Show components of listed themes show the components from a list of themes you select yourself. These themes can include, themes from any other workspace, or themes stored in any other accessible folder. To add a theme select Add… and browse for the theme you wish to add. You can remove any theme from the list by selecting it and selecting Remove.

Figure: Component Store

To restore the settings to those Carbide.ui had when it was first installed select Restore Defaults. Selecting Apply applies the settings without exiting the preferences dialog. If Apply is not used then the settings become active when OK is selected in the main Preferences dialog.

For more information on effectively using the Component Store see Working with the component library.
