Optimizing component properties

  1. In the Editor, click the component you want to optimize.
    The component is highlighted and its properties are listed in the Properties view.
  2. In the Properties view, select the Bitmap tab.
  3. Check the box next to optimize.
  4. Set the quality for the component.
    You can set the quality to:
    • high (16-bit)
    • low (8-bit)
    • palette dependent
  5. If you want to dither the image, check the box next to dither.
    The component properties change in Editor.
  6. To save the changes, select File -> Save.
Note: When you optimize component ptoperties of .svg components in Properties view's Bitmap tab, the next time you save the project, the component is converted into a bitmap. This process is irreversible.
