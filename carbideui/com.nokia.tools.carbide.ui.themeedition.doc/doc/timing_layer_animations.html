Timing Layer Animations

  • In the Editor, right-click the animated component.
  • In the Animation editor, right-click the animated effect.
Ensure that the Animate checkbox is marked in the Properties view for all those values that you want to animate. With the slider for each value, adjust the intensity of the effect at each control point.
  • The animation timing is ready to be set for the chosen values.
  • In the Animation editor, define control points for the animated effect.
Repeat this procedure for every animated effect in your animation.

Note: You can also set static values for effects in the Layers view.
