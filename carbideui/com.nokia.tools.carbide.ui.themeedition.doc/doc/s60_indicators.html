Indicators

The Symbian phone uses different indicators in its UI screen. These indicators are located under Indicators section in the Resources view of Carbide.ui. The Indicators section includes the following sub-sections:

Volume Indicators
These include, for example, the Edited Setting Volume On sub-section.

Tabs Indicators
These include the Active and Passive Tab sub-sections.

Signal/Battery Indicators
These include the signal and battery strength indicators.

Slider Indicators
These include the Setting List Slider and Edited Settings List Slider.

Navigation Indicators
These include the Navi Pane and the Submenu indicators.

Wait/Progress Bar Indicators
These include, for example, the Progress Frames.
