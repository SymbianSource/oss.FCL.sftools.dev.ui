Managing themes

Carbide.ui handles themes as projects. You can import your own theme and theme components into Carbide.ui.

When working with themes, you have the following options:
  • Managing themes
    • This includes creating new and opening existing themes.
  • Importing and exporting themes
  • Managing theme component libraries
    • This includes adding and deleting of, for example, icons to and from icon libraries, one component at a time.
  • Editing theme components
    • This includes editing components in Carbide.ui and in external editors.
  • Creating tasks for themes
    • This includes, for example, creating tasks for certain repeated image component.
  • Deleting themes    
Finalized themes can be compiled into sis-packages. These files can then be sent to a mobile phone or added to your personal theme library. Carbide.ui supports Digital Rights Management (DRM), allowing you to securely protect your themes.

Themes are located in your workspace. You are asked to define a workspace when you open Carbide.ui, but you can always change your workspace from the File menu. Also, different users can have their own workspaces. You can also link existing themes into your workspace.
