Layers View

The Layers view shows all the layers within a component. All components have one layer within their graphic, but some components can have up to three layers. Where multiple layers are available, the final graphic can be defined by using various graphic options on the layer.
Each layer is presented in a tree view form, which can be hidden/unhidden using the expand/collapse icon. Within each layer, the tree may contain:
  • The base layer (usually an image).
  • An optional mask. A mask can be either a soft (white) mask or a hard (black) mask.
  • A list of the colors used in the graphics.

Note: You cannot add new layers through the layers view. You can add, change and remove layers and layer graphics using the Edit/Animate Selection tool button.
