Copying and Pasting elements from the Resources view

Copying elements from the Resources view

When you select elements, sub-elements, or a component group in the Resources view all the components that belong to the hierarchy are selected. The selected components are visible in the Components view. It is then possible to copy all the components for pasting into a different theme.

To copy all the components in a hierarchy, select the desired elements, sub-elements or component group and activate the context menu. It is then possible to copy all the components by selecting Copy Element(s) -> All, as shown in the figure below, or to select only those components which have been customized by selecting Copy Element(s) -> Skinned Only.




Pasting elements to the Resources view

Once a set of components have been copied from a theme, they can be pasted into another theme from the Resources view’s context menu by selecting Paste Elements (into) Theme, as shown in the figure below. Regardless of the category selected in the Resources view, this option pastes all the elements copied from the other theme, into their respective category in the current theme.
