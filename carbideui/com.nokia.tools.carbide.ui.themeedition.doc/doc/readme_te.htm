Read me

Carbide.ui includes a comprehensive help that instructs you in using the tool. The help section is divided into three parts:

  • Active help
    This is context-sensitive help, accessible through a hotkey.
  • Help content
    This is the complete help, accessible through the menu bar.
  • Tutorials
    These include step by step directions in different parts of theme creation.
You can use tutorials to guide you through different tasks with the tool.
To access a tutorial, do one of the following:
  • From the menubar, select Help -> Welcome. On the Welcome page, click Tutorials and then click a tutorial.
  • From the menubar, select Help -> Cheat Sheets and select the desired tutorial.
When working with the tool, you can quickly access help pages on the subject at hand:
  • Click the Help button in windows and wizards to access help pages.
  • Press F1 in views and pop-up windows to open a help page on the subject.
When you start the Carbide.ui tool for the first time, you are taken into a Welcome screen, which serves as a good starting point for getting to know your way around the tool. In the Welcome screen you can, for example, view the user guide, create a theme, open an existing theme, and view the tutorials. You can access the Welcome screen at any time, by selecting in the file menu Help -> Welcome.
