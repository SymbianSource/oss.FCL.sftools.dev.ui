Editing components


You can edit the components and save the edited version of the component. You can also add or remove components. Carbide.ui provides the following options for editing components:
  • Views in the Carbide.ui
    • These include, for example, the Editor and the Outline view.
  • External editors
    • These include the vector graphics, bitmap, and sound editor.
  • Animations
    • Carbide.ui supports animations for certain icons and images.
