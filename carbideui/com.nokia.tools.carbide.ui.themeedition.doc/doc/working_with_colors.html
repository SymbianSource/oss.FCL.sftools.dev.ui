Working with colors

Editing colors


Your theme can specify a variety of font and highlight colors. Carbide .ui allows you to choose a color using several standard color editing styles (palette, RGB, and HSB). Alpha (transparency) values are not supported in themes (all colors are 100% opaque). Each color element can be separately specified for the main and mini handset screen.

Tip: Some standard-level phone models use color rectangles as a highlight (where more advanced-level phones use images). By default, the Workspace and All Screens Display show the images defined for these elements.

If you define values for the color rectangles and want to see how they look in the Workspace or All Screens Display, select the appropriate screen in the Screen Options and set the Row Highlight Type Displayed field to Color.
