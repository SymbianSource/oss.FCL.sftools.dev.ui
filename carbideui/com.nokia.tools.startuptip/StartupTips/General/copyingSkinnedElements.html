How can I copy only the skinned elements from Components view in a single click?

Select a particular group/sub-group in the Resources view. All the elements belonging to it will be shown in the Components view. Now, right click on that particular group/sub-group in the Resources view. Choose Copy Elements -> Skinned only. Skinned only will appear only if one / more elements in the Components view for the particular group/sub-group have been skinned.
