How can I close the themes in a faster way?

If multiple themes are open in the tool, the editor will show only one theme at any point in time. There is a Close ( x ) mark which appears on the right corner of the title bar of the non-active theme. Clicking on this Close ( x ) mark will close the theme without activating it and will consume less time. You can close all the non-active themes in a single operation: Right click on the title bar of Active theme( the one which is displayed in the editor ) and choose Close Others.
